# Dashboard Guide

## Dashboard Sections

1. KPI tiles for executive summary
2. Patient journey funnel
3. Retention trend over time
4. Cost by condition
5. Drop-off analysis
6. Risk segmentation
7. Engagement distribution
8. Insight, action, recommendation, and decision panel

## How to Use
Start with KPI tiles, then move to funnel and retention trend. Use filters to isolate age group, condition, device, region, visit type, and date.
